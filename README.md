# mport

a demo website for a hosting company. it is frontend only.

## stack

- vite + react + typescript
- tailwind css v4
- lenis for the smooth scroll
- react router for the two routes
- inter and playfair display, self-hosted with @fontsource
- phosphor icons

## run it

```bash
pnpm install
pnpm dev
```

## build

```bash
pnpm build      
pnpm preview    
```

## notes

the login screen has no backend, the website is frontend only
